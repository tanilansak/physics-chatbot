3. ชวนคิดตาม Heller & Heller:

  • Understand the Problem: “แสดงแผนภาพวัตถุอิสระสิ วัตถุนี่มีแรงอะไรบ้าง?”

  • Describe the Physics: “กฎของนิวตันข้อไหนใช้ได้ที่นี่?”

  • Plan a Solution: “มีแรงดึงในแนว x เท่าไร? หาแรงลัพธ์ยังไง?”

  • Execute the Plan: “ลองคำนวณดูสิ! ใช้สูตร a = F/m”

  • Evaluate the Answer: “คำตอบสมเหตุสมผลไหม?”

  • “ต้องแยกแรงแนวใด?”

🛠 ขั้นตอนเปิดใช้ GitHub Pages

🔹 ขั้นที่ 1: สมัครบัญชี GitHub (ถ้ายังไม่มี)

  2. คลิก Sign up และกรอกอีเมล รหัสผ่าน ตั้งชื่อผู้ใช้

  3. ยืนยันอีเมล (จะมีลิงก์ส่งไปที่ inbox)

🔹 ขั้นที่ 2: สร้าง Repository (พื้นที่เก็บไฟล์เว็บของคุณ)

  1. หลังเข้าสู่ระบบแล้ว → คลิกปุ่ม ➕ ที่มุมขวาบน → เลือก New repository

    ✅ physics-chatbot (หรืออะไรก็ได้เป็นภาษาอังกฤษ)

  3. เลือก:

    • Public

    • ✅ ติ๊ก Add a README file

  4. กด Create repository

🔹 ขั้นที่ 3: อัปโหลดไฟล์ index.html (จากที่เราสร้างไว้)

  1. คลิกแท็บ “Add file” → “Upload files”

  2. ลากไฟล์ index.html ที่ได้จาก Chatbot ไปใส่

  3. กดปุ่ม “Commit changes” ด้านล่าง

✅ ตอนนี้ chatbot ของคุณอยู่ใน GitHub แล้ว!

🔹 ขั้นที่ 4: เปิดใช้ GitHub Pages

  1. ไปที่หน้า repo ของคุณ → คลิก “Settings”

  2. ที่เมนูซ้าย เลือก “Pages”

  3. ตรง “Source” → เลือก:

    • Branch: main

    • Folder: / (root)

  4. กด Save
